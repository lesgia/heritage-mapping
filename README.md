# Heritage Mapping Project

## Introduction
This repository is dedicated to a project focused on mapping heritage sites and resources.

## Features
- Interactive mapping of heritage sites.
- Data visualization tools for analyzing heritage resources.
- Community contributions for site data.

## Technology Stack
- JavaScript
- Leaflet.js for mapping
- Node.js for backend services

## Usage
To get started, clone the repository and install the necessary dependencies:
```bash
git clone https://github.com/lesgia/heritage-mapping.git
cd heritage-mapping
npm install
```

- Expand the dataset with more heritage sites.
